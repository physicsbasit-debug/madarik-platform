# Phase 8-B: Scientific Diagram Preview and SVG Rendering

## الهدف

تحويل نموذج العقد والروابط إلى رسم SVG فعلي قابل للمعاينة والتنزيل.

## التخطيطات

- العملية: ترتيب عمودي.
- التسلسل والسبب والنتيجة: ترتيب أفقي.
- الدورة: توزيع دائري.
- المقارنة: عمودان.
- التركيب: عقدة رئيسة وفروع.

## عناصر SVG

- عقد مستطيلة.
- أسهم موجهة.
- تسميات اختيارية للروابط.
- عنوان للرسم.
- مساحة عرض ثابتة قابلة للتكبير.

## الجاهزية

يُبلغ النظام عن:

- غياب العنوان.
- غياب العقد.
- روابط تشير إلى عقد غير موجودة.

## خارج النطاق

- PNG.
- PDF.
- تحرير العقد بالسحب.
- توليد الرسوم بالذكاء الاصطناعي.
