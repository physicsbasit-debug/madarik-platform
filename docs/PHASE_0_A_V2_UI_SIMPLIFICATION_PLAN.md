# Phase 0-A: V2 UI Simplification Plan

## يُحذف من المسار الرئيسي

- تكرار عناوين المراحل.
- تكرار معلومات المدرسة والمادة والصف.
- الأزرار الجماعية المبعثرة.
- تفاصيل مزود الذكاء الاصطناعي.
- السجلات التقنية.
- لقطات التخطيط الخام.
- التقارير المتقدمة المفتوحة دائمًا.
- أزرار التصدير المكررة.
- الربط اليدوي الإجباري للرسوم.

## يُنقل إلى قسم متقدم

- بيانات الاستخراج الخام.
- أدوات القص اليدوي.
- تقارير الجودة التفصيلية.
- سجل مزود الذكاء الاصطناعي.
- معلومات JSON.
- الأدوات التشخيصية.

## يبقى في المسار الأساسي

- اختيار المهمة.
- اختيار المصدر.
- ملخص المعالجة.
- التنبيهات المهمة.
- مراجعة الأسئلة.
- الجاهزية.
- التصدير.
