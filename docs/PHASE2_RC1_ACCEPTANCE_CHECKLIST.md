# Phase 2-RC1 Acceptance Checklist

## Backend

- [ ] `pytest -q` ينجح.
- [ ] لا توجد قاعدة بيانات مرفوعة داخل المستودع.
- [ ] endpoints الحسابات تعمل.
- [ ] endpoints المشاريع تعمل.
- [ ] endpoints answer key / analysis / quality tools تعمل.

## Frontend

- [ ] `npm run build` ينجح.
- [ ] لوحة الحسابات تظهر للمالك فقط.
- [ ] مكتبة المشاريع تعمل.
- [ ] رفع PDF لا يكسر المسار.
- [ ] شاشة التصدير تعرض:
  - مسودة نموذج الإجابة.
  - التحليل التربوي.
  - أدوات الجودة.

## Export

- [ ] DOCX يعمل.
- [ ] PDF يعمل.
- [ ] الأسئلة المحذوفة لا تظهر في التصدير.
- [ ] RTL مقبول بصريًا.

## قرار الإغلاق

- [ ] لا توجد أخطاء مانعة.
- [ ] Phase 2-RC1 صالح كمرشح إصدار.
- [ ] أي تحسينات لاحقة تنقل إلى Phase 3 أو Fix محدود.
