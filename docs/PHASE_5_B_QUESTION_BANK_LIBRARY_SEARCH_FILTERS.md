# Phase 5-B: Question Bank Library, Search, and Filters

## الهدف

إضافة مكتبة مستقلة لاستعراض أسئلة البنك والبحث فيها وتصفية النتائج.

## إمكانات البحث

- البحث في النص الأصلي والمترجم.
- البحث برقم السؤال.
- التصفية حسب الصف.
- التصفية حسب المجال العلمي.
- التصفية حسب الوحدة.
- التصفية حسب التصنيف المعرفي.

## واجهة المكتبة

- بطاقة بحث وفلاتر.
- عدد النتائج.
- قائمة بالأسئلة المطابقة.
- معاينة السؤال المحدد.
- عرض التصنيف والصف والدرجة والمشروع المصدر.
- حذف العنصر من البنك.

## الخصوصية

عند توفر حساب مستخدم، تُعرض العناصر التابعة للحساب فقط. العناصر القديمة التي لا تحتوي مالكًا تبقى متاحة وفق سلوك التوافق الحالي.

## خارج النطاق

- نسخ سؤال إلى مشروع آخر.
- إنشاء مشروع من سؤال.
- الاختيار المتعدد.
- إنشاء اختبار.
- مشاركة البنك بين المستخدمين.
- ترتيب النتائج حسب الصعوبة أو الجودة.
