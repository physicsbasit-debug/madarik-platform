# منصة مدارك - حالة Phase 1 حتى N1

## الهدف الحالي

الوصول إلى نسخة تجريبية مستقرة لمسار معالجة ورقة اختبار إنجليزية وتحويلها إلى نسخة عربية/ثنائية اللغة قابلة للمراجعة والتصدير.

## المسار المدعوم حاليًا

1. إنشاء مشروع مؤقت.
2. إدخال بيانات الورقة.
3. رفع PDF نصي أو صورة أو PDF مصوّر محدود الصفحات.
4. استخراج النص.
5. تقسيم النص إلى بطاقات أسئلة.
6. توليد قاموس مصطلحات أولي.
7. ترجمة أولية مع fallback آمن.
8. مراجعة الأسئلة وتعديل الترجمة والدرجات.
9. ربط مرفقات يدوية بالأسئلة.
10. تنفيذ إجراءات جماعية على حالات الأسئلة.
11. فحص جاهزية التصدير.
12. تصدير DOCX/PDF.
13. حفظ واستيراد نسخة عمل JSON.

## ما لا يزال مؤجلًا

- الحسابات والصلاحيات.
- الحفظ الدائم في قاعدة بيانات.
- سجل التعديلات.
- نموذج الإجابة.
- OCR عربي.
- استخراج الصور والجداول تلقائيًا من PDF.
- تحليل تربوي للأسئلة.
- تشغيل AI خارجي إلزامي.

## ملاحظة تشغيلية

هذه النسخة مناسبة لاختبار قبول داخلي بملفات حقيقية، وليست بعد نسخة إنتاج مدرسية نهائية.
