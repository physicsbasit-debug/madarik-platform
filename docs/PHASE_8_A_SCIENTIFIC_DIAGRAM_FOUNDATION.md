# Phase 8-A: Scientific Diagram Data Model and Workspace Foundation

تضيف المرحلة نموذج بيانات للرسوم العلمية، مع عقد وروابط واتجاهات، وأنواع تشمل
العملية والدورة والمقارنة والتسلسل والتركيب والسبب والنتيجة. تحفظ الرسوم في
SQLite وتدار من مساحة عمل RTL. لا تتضمن هذه المرحلة توليدًا ذكيًا أو SVG نهائيًا.
