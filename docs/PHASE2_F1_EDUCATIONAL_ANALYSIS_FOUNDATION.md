# منصة مدارك - Phase 2-F1 Educational Analysis Foundation

## الغرض

إضافة تحليل تربوي تأسيسي للورقة بعد استخراج الأسئلة ومراجعتها، بهدف إعطاء المعلم قراءة أولية عن اتزان الورقة قبل التصدير.

## ما تم

- إضافة نموذج:
  - `EducationalAnalysisReport`
- إضافة حقل داخل المشروع:
  - `educational_analysis`
- إضافة خدمة:
  - `backend/app/services/educational_analysis.py`
- إضافة endpoints:
  - `POST /api/projects/{project_id}/educational-analysis`
  - `DELETE /api/projects/{project_id}/educational-analysis`
- تحليل تأسيسي يشمل:
  - عدد الأسئلة النشطة.
  - مجموع الدرجات.
  - متوسط الدرجة لكل سؤال.
  - توزيع أوامر السؤال.
  - توزيع الدرجات.
  - عدد الأسئلة المترجمة.
  - وجود مسودة نموذج الإجابة.
  - وجود لقطات تخطيط PDF.
  - تنبيهات وتوصيات للمعلم.
- عرض التحليل في خطوة التصدير.

## حدود المرحلة

- لا يوجد تحليل نفسي قياسي.
- لا يوجد Pareto أو Fishbone أو Radar بعد.
- لا يوجد ربط مهارات رسمي.
- لا يوجد تحليل جودة عميق للورقة.
- لا يصدر التحليل داخل ملف الطالب.

## معيار القبول

- توليد التحليل من الأسئلة النشطة.
- تجاهل الأسئلة المحذوفة.
- حفظ `educational_analysis` داخل المشروع.
- عرض الخلاصة والتوصيات والتنبيهات في الواجهة.
- حذف التحليل.
- بقاء اختبارات Backend وFrontend خضراء.
