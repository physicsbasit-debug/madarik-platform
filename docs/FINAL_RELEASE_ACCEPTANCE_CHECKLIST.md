# قائمة القبول النهائي للإصدار

## أولا: البوابة الآلية

- [ ] `RUN_PHASE10_A_RELEASE_AUDIT.py` ناجح.
- [ ] لا SQLite أو ZIP أو Patch أو ملفات bytecode متتبعة.
- [ ] لا ملفات `.env` أو مفاتيح أو شهادات متتبعة.
- [ ] الإصدار متطابق بين README وBackend وFrontend وpackage-lock.
- [ ] OpenAPI دون `operationId` مكرر.
- [ ] `/api/health/readiness` يرجع `technical_ready=true`.
- [ ] `python -m compileall -q backend/app backend/tests` ناجح.
- [ ] اختبارات Phase 10-A المركزة ناجحة.
- [ ] اختبارات Backend الكاملة ناجحة.
- [ ] `npm ci` ناجح.
- [ ] `npm run lint` ناجح.
- [ ] `npm run build` ناجح.
- [ ] GitHub Actions خضراء على الفرع أو Pull Request.

## ثانيا: القبول الحي لمزود الترجمة

- [ ] مزود خارجي حقيقي مفعل عبر متغيرات البيئة فقط.
- [ ] اسم نموذج حقيقي ومدعوم من لوحة المزود.
- [ ] ترجمة ورقة Cambridge كاملة.
- [ ] `used_external_provider=true` للنتيجة المقبولة.
- [ ] لا توجد نتيجة `fallback` معتمدة.
- [ ] لا نثر إنجليزي غير مفسر في النسخة العربية.
- [ ] القاموس المعتمد مطبق.
- [ ] الأرقام والوحدات والمعادلات والرموز العلمية محفوظة.
- [ ] محاولة التصحيح الواحدة تعمل عند الحاجة ولا تدخل حلقة إعادة.

## ثالثا: المراجعة البصرية

### DOCX

- [ ] يفتح دون رسالة إصلاح.
- [ ] RTL والمحاذاة صحيحة.
- [ ] الأسئلة والأجزاء والخيارات مرتبة.
- [ ] الرسوم والجداول موجودة وواضحة.
- [ ] الدرجات والمجموع متسقان.
- [ ] الأسس والوحدات والرموز سليمة.
- [ ] لا تظهر ملاحظات داخلية أو placeholders.

### PDF

- [ ] يفتح دون أخطاء.
- [ ] RTL والمحاذاة صحيحة.
- [ ] لا قص أو تداخل أو فقد صفحات.
- [ ] الرسوم والجداول موجودة وواضحة.
- [ ] الدرجات والمجموع متسقان.
- [ ] الأسس والوحدات والرموز سليمة.
- [ ] لا تظهر ملاحظات داخلية أو placeholders.

## رابعا: قرار الإصدار

- [ ] لا توجد مشكلة `FIX REQUIRED`.
- [ ] لا يوجد `LIVE BLOCKER` مفتوح.
- [ ] محضر `docs/FINAL_RELEASE_ACCEPTANCE.md` مملوء بالأدلة.
- [ ] جرى الدمج في `main` بعد نجاح البوابة.
- [ ] أنشئ Tag بعد القبول فقط.
- [ ] أنشئ GitHub Release بعد القبول فقط.

## الحكم الحالي بعد Phase 10-A

- البوابة التقنية: قابلة للإغلاق بعد نجاح GitHub Actions.
- الإصدار الإنتاجي: غير معتمد قبل القبول الحي والمراجعة البصرية.
