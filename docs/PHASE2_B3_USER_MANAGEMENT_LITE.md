# منصة مدارك - Phase 2-B3 User Management Lite

## الغرض

إضافة إدارة حسابات خفيفة لحساب المالك، فوق طبقة الحسابات والصلاحيات التي بدأت في Phase 2-B1 وربط المشاريع الذي بدأ في Phase 2-B2.

## ما تم

- إضافة endpoint لعرض الحسابات:
  - `GET /api/auth/accounts`
- إضافة endpoint لإنشاء حساب:
  - `POST /api/auth/accounts`
- إضافة endpoint لتحديث الحساب:
  - `PATCH /api/auth/accounts/{account_id}`
- واجهة مالك المنصة تستطيع:
  - عرض الحسابات.
  - إنشاء حساب معلم أو مراجع أو مالك.
  - تفعيل/تعطيل حساب.
- حماية إدارة الحسابات بحيث تكون للمالك فقط.
- منع المالك من تعطيل نفسه.
- منع حذف/تعطيل آخر مالك نشط.

## ما لم يدخل

- حذف حساب نهائي.
- تغيير كلمة المرور من الواجهة.
- نقل ملكية المشاريع.
- دعوات بريدية.
- صلاحيات تفصيلية لكل مشروع.
- سجل تدقيق للحسابات.

## ملاحظة

هذه ليست إدارة مستخدمين كاملة، بل طبقة تشغيلية خفيفة تكفي لاختبار تعدد الحسابات وربطها بالمشاريع في المراحل التالية.
