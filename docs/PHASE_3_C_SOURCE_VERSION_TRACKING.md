# Phase 3-C: Source Version Tracking and Refresh Detection

## الهدف

كشف تغير ملفات Google Drive المرتبطة بالمناهج دون تحديثها أو استبدالها تلقائيًا.

## الحالات

- `unknown`: لم يُفحص.
- `current`: مطابق للنسخة المرتبطة.
- `changed`: تغير checksum أو تاريخ التعديل.
- `missing`: اختفى من المجلد المسموح.
- `unverifiable`: تعذر الاتصال أو نقصت بيانات المقارنة.

## ترتيب المقارنة

1. checksum عند توفره في النسختين.
2. تاريخ التعديل عند غياب checksum.
3. غير قابل للتحقق عند غياب المؤشرين.

## الحماية

- الفحص يغير حالة Metadata فقط.
- لا ينزل الملف تلقائيًا.
- لا يستبدل المحتوى.
- لا يحذف المصدر المفقود.
- القرار النهائي للمستخدم.

## مؤجل

- قبول نسخة جديدة.
- مزامنة دورية.
- تنبيه خلفي.
- إعادة تحليل المحتوى والأسئلة.
