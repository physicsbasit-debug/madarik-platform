# منصة مدارك - Release Notes RC1

## الإصدار

Phase 1-RC1  
Version: `1.0.0-rc.1`

## وصف الإصدار

هذه نسخة مرشحة أولى لاختبار قبول Phase 1. هدفها تشغيل دورة معالجة ورقة اختبار إنجليزية وتحويلها إلى نسخة عربية/ثنائية اللغة قابلة للمراجعة والتصدير.

## القدرات المدعومة

- إنشاء مشروع مؤقت.
- إدخال بيانات الورقة.
- رفع PDF نصي.
- رفع صورة PNG/JPG/WEBP وتشغيل OCR إنجليزي مبدئي.
- محاولة OCR على PDF مصوّر محدود الصفحات.
- تقسيم النص إلى بطاقات أسئلة.
- توليد قاموس مصطلحات أولي.
- ترجمة أولية مع fallback آمن.
- مراجعة السؤال وتعديل الترجمة والدرجة.
- حذف/استعادة/ترتيب الأسئلة.
- ربط مرفقات يدوية بالأسئلة.
- إجراءات جماعية في شاشة المراجعة.
- فحص جاهزية قبل التصدير.
- تصدير DOCX وPDF بتنسيق RTL أولي.
- حفظ واستيراد نسخة عمل JSON.

## خارج نطاق RC1

- الحسابات والصلاحيات.
- قاعدة بيانات دائمة.
- حفظ سحابي.
- نموذج إجابة.
- OCR عربي.
- استخراج الصور والجداول تلقائيًا من PDF.
- تحليل تربوي للأسئلة.
- تشغيل AI خارجي إلزامي.
- اعتماد الإنتاج المدرسي النهائي.

## معيار القبول

يُقبل RC1 إذا:

- GitHub Actions أخضر.
- Backend tests ناجحة.
- Frontend build ناجح.
- اختبار قبول يدوي ناجح وفق `PHASE1_ACCEPTANCE_PROTOCOL.md`.
- فتح DOCX/PDF الناتجين بصريًا وتأكيد اتجاه RTL المقبول.
