# منصة مدارك - Phase 2-RC1 Release Notes

## الحالة

Phase 2-RC1 هي نسخة مرشحة لتثبيت نطاق المرحلة الثانية. لا تضيف هذه المرحلة ميزة تشغيلية جديدة، بل تغلق النطاق وتوثق نقاط القبول ومسار الاختبار. نعم، أحيانًا أفضل ميزة هي التوقف عن إضافة ميزات، وهي فكرة تبدو للبشر غريبة على نحو مؤلم.

## ما يغطيه Phase 2

### التخزين والمكتبة
- SQLite persistence foundation.
- مكتبة المشاريع.
- Snapshot JSON بقي كمسار احتياطي.

### الحسابات والصلاحيات
- حساب مالك أولي.
- تسجيل الدخول والخروج.
- أدوار `owner`, `teacher`, `reviewer`.
- ربط المشاريع بحساب المالك/المستخدم.
- إدارة حسابات خفيفة للمالك.

### الذكاء الاصطناعي
- طبقة AI Provider آمنة.
- دعم `mock`, `openai`, `openai-compatible`.
- إعدادات hardening وfallback آمن.
- عدم كشف مفاتيح API.

### معالجة PDF والأصول
- استخراج نص PDF/OCR كما في Phase 1.
- لقطات تخطيط PDF داخل `layout_assets`.

### مسودة الإجابة والتحليل
- `answer_key` لمسودة نموذج الإجابة.
- `educational_analysis` للتحليل التربوي التأسيسي.
- `quality_tools` لأدوات Pareto/Radar/Fishbone التأسيسية.

## ما لا يدخل في RC1

- رسوم تفاعلية حقيقية.
- تصدير التحليل وأدوات الجودة داخل PDF/DOCX.
- Rubric تفصيلي.
- ربط مهارات رسمي بمعايير المنهج.
- مشاركة المشاريع بين الحسابات.
- إدارة كلمات مرور متقدمة.
- نشر إنتاجي كامل.

## معيار القبول

- Backend tests خضراء.
- Frontend build أخضر.
- فتح مشروع جديد.
- تسجيل دخول مالك.
- إنشاء مشروع محفوظ.
- رفع PDF.
- استخراج أسئلة.
- توليد قاموس وترجمة.
- توليد مسودة نموذج إجابة.
- توليد تحليل تربوي.
- توليد أدوات جودة.
- تصدير DOCX/PDF.
