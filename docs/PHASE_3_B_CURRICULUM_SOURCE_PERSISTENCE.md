# Phase 3-B: Persist Imported Sources and Attach Them to Curriculum

## الهدف

حفظ مصدر Google Drive داخل مشروع مدارك وربطه بالسياق المنهجي المختار.

## البيانات المحفوظة

- مزود المصدر.
- معرف الملف.
- اسم الملف ونوعه وحجمه.
- checksum عند توفره.
- الصف والمجال العلمي.
- الفصل والمادة والوحدة.
- نوع الوثيقة.
- تاريخ الاستيراد وتاريخ تعديل المصدر.

## منع التكرار

تُمنع النسخة المكررة عند تطابق:

- provider
- source file ID
- checksum

## واجهة المستخدم

- عرض المصادر المرتبطة بالسياق الحالي.
- استيراد وربط.
- تعطيل الملف المرتبط.
- حذف الربط دون حذف الملف الأصلي.

## خارج النطاق

- حفظ bytes الملف داخل المشروع.
- التحليل التلقائي بعد الربط.
- مزامنة الإصدارات.
- ربط الأسئلة بالمصدر.
