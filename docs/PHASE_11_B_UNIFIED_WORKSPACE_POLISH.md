# Phase 11-B: Unified Workspace Polish and Visual Acceptance

## الهدف

تثبيت لغة بصرية واحدة لجميع وحدات منصة مدارك بعد نجاح إعادة توجيه المنتج في
Phase 11-A ونشر الواجهة الجديدة على GitHub Pages.

لا تضيف هذه المرحلة وظائف تعليمية أو عقود API جديدة. نطاقها يقتصر على وضوح
الواجهة، كثافة المعلومات، الاستجابة للشاشات، الاتساق، وإتاحة الاستخدام.

## ما تم تحسينه

### غلاف المنصة

- تقليل عرض الشريط الجانبي وكثافة عناصره دون إخفاء أسماء الوحدات.
- تحسين تدرج أحجام النص والأيقونات ومجموعات التنقل.
- ضغط بطاقتي حالة الاتصال والحساب في أسفل الشريط.
- إضافة عنوان واضح لإجراء «المزيد» بدل زر سهم مبهم.
- توحيد حالات التركيز بلوحة المفاتيح لجميع العناصر التفاعلية.

### رأس مساحة العمل

- تقليل الارتفاع والفراغات غير الضرورية.
- تحسين توازن عنوان القسم، المشروع النشط، وإجراءات المشروع.
- إضافة حاوية محتوى مركزية بحد أقصى يمنع تمدد الصفحات بصورة مفرطة على الشاشات الكبيرة.

### لوحة التحكم

- تقليل ارتفاع البطاقة البطولية مع الحفاظ على وضوح الرسالة الأساسية.
- تحسين نسبة بطاقة المشروع النشط إلى البطاقة الرئيسية.
- ضغط مؤشرات الأداء لتظهر بصورة أفضل فوق الطية الأولى.
- تقليل ارتفاع بطاقات الوحدات وتوحيد المسافات والنصوص.
- إعادة صياغة النصوص بصوت مهني مباشر وخالٍ من العبارات المزاحمة للعمل.

### الوحدات الرئيسية

تم تطبيق نمط رأس موحد على:

- المصادر السحابية.
- المناهج والدروس.
- بنك الأسئلة.
- منشئ الاختبارات.
- الأنشطة التعليمية.
- الرسوم العلمية.
- الترجمة السريعة.

يشمل النمط الموحد: الحواف، الظلال، الخلفية، حجم العنوان، الوصف، وزر الرجوع.

### الاستجابة للشاشات

- شريط جانبي مكتبي مدمج حتى عرض 1120 بكسل.
- قائمة منزلقة للأجهزة اللوحية والأصغر.
- تحويل بطاقات الوحدات والمؤشرات إلى عمودين ثم عمود واحد حسب المساحة.
- إخفاء نصوص الإجراءات الثانوية فقط عندما لا تتسع الشاشة، مع إبقاء الأيقونات وأسماء الإجراءات الأساسية.

## ما لم يتغير

- Backend وOpenAPI.
- نماذج البيانات والتخزين.
- Gemini والترجمة العلمية.
- OCR ومعالجة PDF.
- التصدير إلى DOCX وPDF وPNG وSVG.
- وظائف المصادر والمناهج وبنك الأسئلة والاختبارات والأنشطة والرسوم.

## القبول البصري

يعتمد القبول النهائي لهذه المرحلة على فحص الواجهة المنشورة في الأحجام التالية:

- 1440×900 للحاسب.
- 1280×720 للشاشات المتوسطة.
- 1024×768 للجهاز اللوحي.
- 390×844 للهاتف.

ويشمل الفحص الصفحات التسع الرئيسية والتنقل بينها دون قص نصوص أو تداخل أو تمرير أفقي غير مقصود.
