# Phase 4-A5: Translation Acceptance & Batch Reliability

## الهدف

تحويل مسار الترجمة من معالجة متتابعة قد تتوقف عند خطأ واحد إلى دفعة موثوقة تعزل كل سؤال وكل جزء، وتحفظ النتائج الناجحة، وتستمر حتى نهاية الورقة.

## نطاق المرحلة

- ترجمة كل سؤال مستقل أو جزء مستقل بوصفه عنصرًا داخل الدفعة.
- تجاوز الأسئلة المحذوفة دون إرسالها إلى أي مزود خارجي.
- الحفاظ على الأجزاء الفارغة والعناوين الهرمية دون محاولة ترجمتها.
- عزل الأخطاء غير المتوقعة في المزود أو fallback المحلي لكل عنصر.
- عدم إسقاط الأسئلة الأخرى عند فشل عنصر واحد.
- حفظ ملخص الدفعة داخل `ProjectSession` وSQLite وملفات Snapshot.
- عرض ملخص الدفعة في شاشة المراجعة.
- إضافة تحذيرات جاهزية عند استخدام fallback أو وجود فشل محفوظ بأمان.
- إبقاء `translate_questions_with_glossary` متوافقًا مع الاستدعاءات القديمة.

## تصنيفات نتيجة العنصر

- `external_success`: نجحت الترجمة الخارجية من المحاولة الأولى.
- `corrected_success`: نجحت الترجمة الخارجية بعد محاولة التصحيح الواحدة.
- `local_fallback`: استُخدمت الترجمة المحلية الآمنة.
- `skipped`: عنصر محذوف أو فارغ لا يحتاج إلى ترجمة.
- `failed_safely`: تعذر المزود وfallback أو حدث خطأ في تنسيق العنصر، فحُفظ دون إسقاط الدفعة.

## حالة الدفعة

- `completed`: لا توجد حالات fallback أو فشل محفوظ.
- `completed_with_fallbacks`: اكتملت الدفعة مع عنصر أو أكثر باستخدام fallback المحلي.
- `completed_with_failures`: اكتملت الدفعة مع عنصر أو أكثر مصنف `failed_safely`.

## ملخص الدفعة المحفوظ

يتضمن الملخص:

- عدد الأسئلة الكلي والنشط والمحذوف.
- عدد عناصر الترجمة الفعلية.
- النجاح الخارجي المباشر.
- النجاح بعد التصحيح.
- fallback المحلي.
- العناصر المتجاوزة.
- الفشل المحفوظ بأمان.
- عدد العناصر التي تحتاج مراجعة عاجلة.
- سجل مختصر لكل سؤال أو جزء.

## قرارات السلامة

- لا يعاد إرسال الدفعة كاملة عند فشل عنصر واحد.
- لا توجد إعادة محاولة إضافية فوق محاولة التصحيح الواحدة المعتمدة في Phase 4-A4.
- التعديل اليدوي للسؤال أو القاموس أو بيانات المشروع يبطل ملخص الدفعة القديم حتى لا تعرض الواجهة نتيجة قديمة.
- `failed_safely` لا يحذف النص القديم؛ يحفظ الترجمة السابقة إن وجدت، وإلا يعرض النص المصدر ضمن تنبيه مراجعة عاجلة.
- مراجعة المعلم تبقى إلزامية قبل التصدير.
