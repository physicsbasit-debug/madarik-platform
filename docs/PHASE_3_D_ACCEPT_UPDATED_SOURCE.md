# Phase 3-D: Accept Updated Source and Preserve Version History

## الهدف

اعتماد النسخة الجديدة يدويًا بعد اكتشاف تغير المصدر، مع حفظ Metadata النسخة السابقة.

## عند الاعتماد

1. تُضاف النسخة السابقة إلى سجل الإصدارات.
2. تُحدّث Metadata المصدر المرتبط.
3. تتحول الحالة إلى `current`.
4. يُسجل وقت الفحص والاعتماد.
5. لا تُحذف النسخة السابقة.

## المحفوظ في سجل النسخ

- checksum
- الحجم
- اسم الملف
- MIME type
- تاريخ تعديل المصدر
- تاريخ تسجيل النسخة

## الحماية

- الاعتماد يدوي فقط.
- لا استبدال صامت.
- لا إعادة تحليل للمحتوى.
- لا استعادة نسخة قديمة في هذه المرحلة.
