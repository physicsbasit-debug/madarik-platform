# Phase 6-E: Native DOCX and PDF Assessment Export

## الهدف

إنتاج ملفات اختبار أصلية قابلة للطباعة بدل ملفات الأساس النصية.

## DOCX

يتضمن:

- اتجاه RTL.
- عنوان الاختبار.
- الصف والزمن والدرجة.
- الأقسام والتعليمات.
- الأسئلة والدرجات.
- مساحات الإجابة.
- صفحة إجابة منفصلة.

## PDF

يتضمن نفس البنية العامة في ملف PDF أصلي باستخدام ReportLab.

## الحماية

لا يتم إنشاء ملف نهائي إذا كانت المسودة غير جاهزة. يعاد رد 409 مع قائمة
المشكلات بدل تنزيل ملف ناقص.

## الواجهة

تُنزل الملفات مباشرة من المتصفح عند نجاح التصدير.

## خارج النطاق

- شعار المدرسة.
- رأس قابل للتخصيص.
- نموذج إجابة تفصيلي.
- مفاتيح تصحيح.
