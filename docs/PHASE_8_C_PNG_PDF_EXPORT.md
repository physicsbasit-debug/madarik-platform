# Phase 8-C: Scientific Diagram PNG and PDF Export

## الهدف

إغلاق مسار الرسوم العلمية بتصدير PNG وPDF حقيقيين اعتمادًا على SVG نفسه.

## التنفيذ

- SVG هو المصدر الوحيد للرسم.
- CairoSVG يحول SVG إلى PNG أو PDF.
- تُحفظ الملفات في مجلد exports.
- يبدأ التنزيل مباشرة من المتصفح.

## الجاهزية

يمنع التصدير عند وجود:

- عنوان مفقود.
- عقد مفقودة.
- روابط غير صالحة.

## خارج النطاق

- تحرير بالسحب.
- توليد ذكي للرسم.
- قوالب مؤسسية.
