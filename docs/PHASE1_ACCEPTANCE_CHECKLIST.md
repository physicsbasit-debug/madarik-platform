# منصة مدارك - قائمة اختبار قبول Phase 1

استخدم هذه القائمة قبل تثبيت نسخة RC1.

## 1. بيانات الورقة

- [ ] إدخال اسم المدرسة.
- [ ] إدخال المادة والصف والفصل.
- [ ] اختيار نوع النسخة: عربية أو ثنائية اللغة.
- [ ] رفع شعار المدرسة اختياريًا.

## 2. رفع الملفات والاستخراج

- [ ] رفع PDF نصي واضح.
- [ ] رفع صورة PNG/JPG/WEBP واضحة.
- [ ] تجربة PDF مصوّر بسيط.
- [ ] التأكد أن النص المستخرج يظهر في خطوة الاستخراج.

## 3. الأسئلة والقاموس

- [ ] تحويل النص إلى بطاقات أسئلة.
- [ ] توليد قاموس الورقة.
- [ ] تعديل مصطلح في القاموس.

## 4. المراجعة

- [ ] تعديل ترجمة سؤال.
- [ ] تعديل درجة سؤال.
- [ ] حذف سؤال ثم استعادته.
- [ ] إعادة ترتيب سؤال.
- [ ] إضافة مرفق صورة/جدول لسؤال.
- [ ] استخدام اعتماد كل الأسئلة النشطة.
- [ ] استخدام استعادة الكل للمراجعة.

## 5. الجاهزية والتصدير

- [ ] تحديث فحص الجاهزية.
- [ ] التأكد من منع التصدير عند حذف كل الأسئلة.
- [ ] تصدير DOCX.
- [ ] تصدير PDF.
- [ ] فتح الملفين بصريًا والتأكد من RTL.

## 6. حفظ العمل

- [ ] تنزيل نسخة JSON.
- [ ] إنشاء مشروع جديد.
- [ ] استيراد نسخة JSON.
- [ ] التأكد أن الأسئلة والقاموس والمرفقات عادت.
