# منصة مدارك - Blueprint مختصر

منصة مدارك تعالج أوراق الاختبارات الأجنبية وتحولها إلى موارد عربية وثنائية اللغة قابلة للمراجعة والطباعة.

## المسار الحالي

```text
PDF نصي
→ استخراج النص
→ تقسيم الأسئلة
→ توليد قاموس الورقة
→ ترجمة أولية محافظة
→ مراجعة المعلم
→ تصدير DOCX/PDF
```

## Phase 1-F2

أضافت المرحلة تصدير PDF فعليًا بتنسيق RTL أولي، مع الإبقاء على DOCX كخيار قابل للتعديل.

## حدود المرحلة

- لا OCR.
- لا AI خارجي.
- لا صور أو جداول داخل التصدير.
- لا شعار مدرسة داخل التصدير.
- لا حفظ دائم.
