# MADARIK_BLUEPRINT - منصة مدارك

منصة مدارك منصة تعليمية ذكية لمعالجة أوراق الاختبارات الأجنبية وتحويلها إلى موارد عربية وثنائية اللغة قابلة للمراجعة والطباعة.

## المسار الحالي حتى Phase 1-F1

PDF نصي → استخراج نص → تقسيم أسئلة → قاموس مصطلحات → ترجمة أولية → مراجعة → تصدير DOCX.

## نطاق Phase 1-F1

- إنشاء Word حقيقي.
- دعم RTL في المحتوى العربي.
- دعم النسخة الثنائية بإنجليزي ثم عربي.
- رأس ورقة رسمي.
- حذف الأسئلة المحذوفة من التصدير.

## خارج النطاق

- PDF.
- OCR.
- صور وجداول.
- AI خارجي.
- نموذج إجابة.
