# منصة مدارك - المخطط التفصيلي v0.1

منصة مدارك منصة تعليمية ذكية لمعالجة أوراق الاختبارات والمحتوى التعليمي الأجنبي وتحويله إلى موارد عربية وثنائية اللغة قابلة للمراجعة والتعديل والطباعة.

## نطاق Phase 1

- مواد البداية: علوم عامة، فيزياء، كيمياء، أحياء.
- الملفات العملية الأولى: PDF وصور.
- الإخراج: DOCX وPDF.
- النسخ: عربية نظيفة، وثنائية اللغة نظيفة.
- الترجمة: فصحى تعليمية مباشرة مرتبطة قدر الإمكان بمصطلحات سلطنة عمان.
- المراجعة: بطاقات أسئلة مستقلة، مع تعديل وحذف وترتيب ودرجات.

## ما يؤجل

- نموذج الإجابة.
- النسخة التدريبية المشروحة.
- تصنيف المهارة والصعوبة.
- الحسابات والصلاحيات.
- حفظ المشاريع الدائم.
