# MADARIK BLUEPRINT - v0.5 Phase 1-E2

منصة مدارك منصة تعليمية ذكية لمعالجة أوراق الاختبارات الأجنبية وتحويلها لاحقًا إلى موارد عربية وثنائية اللغة قابلة للمراجعة والطباعة.

## المسار الحالي المثبت

```text
رفع PDF نصي
→ استخراج النص
→ تحويل النص إلى بطاقات أسئلة
→ توليد قاموس مصطلحات أولي للمعلم
→ مراجعة الأسئلة والمصطلحات
```

## المرحلة الحالية

**Phase 1-E2: Glossary Engine**

تضيف هذه المرحلة محركًا أوليًا لاستخراج المصطلحات من بطاقات الأسئلة باستخدام قائمة مصطلحات علمية داخلية. لا تعتمد على AI، ولا تولد ترجمة الأسئلة بعد.

## الهدف القادم

**Phase 1-E2: Translation Engine**

- ترجمة كل سؤال على حدة.
- استخدام قاموس أوامر الأسئلة.
- استخدام قاموس الورقة المعتمد.
- إبقاء الترجمة قابلة للتعديل.

## ما لا يزال مؤجلًا

- OCR.
- ترجمة النصوص داخل الصور.
- ربط الصور والجداول بالسؤال.
- تصدير DOCX/PDF.
- حفظ المشاريع دائمًا.
- حسابات وصلاحيات.
