# Phase 4-A4: Scientific Translation Fidelity Guard

## الهدف

إضافة فحص حتمي بعد الترجمة يحمي المحتوى العلمي الذي لا يجوز للمزود تغييره أثناء نقل السؤال من الإنجليزية إلى العربية.

## العناصر المحمية

- الدرجات مثل `[2]` وصيغ `2 marks`.
- المعادلات والعلاقات مثل `V = IR` و`I = 2 A`.
- القيم المقترنة بوحدات مثل `5 kg` و`20 N` و`10 m/s²`.
- الصيغ الكيميائية البسيطة مثل `H₂O` و`CO2` و`NaCl`.
- القيم العددية والنسب والنسب المئوية والصيغ العلمية.
- رموز أجزاء الأسئلة مثل `(b)(ii)`.
- مراجع الأشكال والجداول مع السماح بترجمة كلمة المرجع وإبقاء المعرّف.
- الرموز العلمية والمتغيرات الواضحة في السياق الرياضي.

## دورة الحماية

1. يستخرج النظام العناصر المحمية من النص المصدر بصورة محافظة.
2. يرسلها إلى المزود داخل قسم `PROTECTED SOURCE CONTENT`.
3. يفحص الناتج آليًا بعد الترجمة مع السماح بفروق المسافات والتمثيل الطباعي غير المؤثر.
4. يجمع مخالفات القاموس ومخالفات الأمان العلمي في محاولة تصحيح واحدة فقط.
5. يعيد الفحص بعد التصحيح.
6. إذا بقيت أي مخالفة، يستخدم الترجمة المحلية الآمنة ويضيف ملاحظة مراجعة واضحة.

## قرارات السلامة

- لا توجد حلقة إعادة محاولة مفتوحة. الحد الأقصى طلبان: ترجمة أولى وتصحيح واحد.
- الكلمات الإنجليزية العادية لا تعامل تلقائيًا كصيغ أو متغيرات.
- الفحص لا يدّعي فهم المعنى الكامل، بل يحمي العناصر القابلة للتحقق الحتمي.
- مراجعة المعلم تبقى إلزامية قبل التصدير.
