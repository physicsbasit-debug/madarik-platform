# قرارات منصة مدارك

## القرارات الثابتة حتى Phase 1-F1

- الاسم: منصة مدارك.
- التقنية: React + TypeScript للواجهة، FastAPI + Python للخلفية.
- التخزين: جلسة مؤقتة داخل الذاكرة في المراحل الأولى.
- PDF نصي فقط في Phase 1-C، وOCR مؤجل.
- تقسيم الأسئلة في Phase 1-D مبدئي ويحتاج مراجعة بشرية.
- القاموس في Phase 1-E1 للمعلم فقط.
- الترجمة في Phase 1-E2 أولية ومحافظة دون API خارجي.
- التصدير في Phase 1-F1 يبدأ بـ DOCX فقط.
- PDF مؤجل لمرحلة مستقلة بسبب حساسية RTL.
- الأسئلة المحذوفة لا تدخل في التصدير.
- الترقيم النهائي يعاد حسب ترتيب المعلم.
