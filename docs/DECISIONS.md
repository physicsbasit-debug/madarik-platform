# قرارات منصة مدارك

## Phase 1-G1: AI Provider Layer

تم اعتماد طبقة مزود ترجمة اختيارية. يبقى `mock` هو الوضع الافتراضي حتى لا تحتاج GitHub Actions أو التطوير المحلي إلى مفاتيح سرية.

عند ضبط مزود خارجي من الخادم فقط، يمكن للترجمة استخدامه. عند أي خطأ أو نقص في الإعدادات، يعود النظام تلقائيًا إلى fallback قابل للمراجعة.

## قواعد ثابتة

- لا تُحفظ مفاتيح API داخل الكود.
- لا تُرسل مفاتيح API إلى المتصفح.
- لا تفشل ترجمة الأسئلة بسبب غياب المزود الخارجي.
- تبقى مراجعة المعلم شرطًا قبل التصدير.

## قرار Phase 1-H1

تُدعم مرفقات السؤال يدويًا فقط في هذه المرحلة. يستطيع المعلم رفع صورة أو جدول لكل سؤال أثناء المراجعة، وتظهر المرفقات في التصدير النهائي. لا تتم محاولة استخراج الصور تلقائيًا من PDF حتى لا نخلط OCR وتحليل التخطيط مع مسار التصدير المستقر.
