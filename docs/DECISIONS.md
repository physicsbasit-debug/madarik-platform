# Decisions - منصة مدارك

## قرارات ثابتة حتى Phase 1-E2

- الاسم المعتمد: منصة مدارك.
- البداية بمواد العلوم والفيزياء والكيمياء والأحياء، دون حصر الاسم في العلوم.
- الواجهة React + TypeScript.
- الخلفية Python + FastAPI.
- لا توجد حسابات في Phase 1.
- حفظ المشروع مؤقت داخل الجلسة فقط.
- التصدير الحقيقي مؤجل.
- OCR مؤجل.
- الترجمة عبر AI مؤجلة إلى Phase 1-E2.
- نموذج الإجابة مؤجل.
- النسخة التدريبية المشروحة مؤجلة.

## قرار Phase 1-E2

- توليد قاموس الورقة يتم بمحرك Rule-based داخلي.
- القاموس يستخرج مصطلحات من بطاقات الأسئلة لا من الصور.
- جميع المصطلحات المكتشفة تظهر بحالة `needs_review` حتى يراجعها المعلم.
- `source=detected` يميز المصطلحات المستخرجة من أسئلة الورقة.
- القاموس لا يظهر للطالب في النسخة النهائية.
- القاموس لا يستخدم بعد في الترجمة؛ هذا مؤجل إلى Phase 1-E2.
