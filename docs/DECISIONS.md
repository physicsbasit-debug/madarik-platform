# قرارات منصة مدارك

## قرارات مثبتة

- اسم المشروع: منصة مدارك.
- الواجهة: React + TypeScript.
- الخلفية: Python + FastAPI.
- اتجاه الواجهة والتصدير: RTL.
- Phase 1-A واجهة ثابتة فقط، دون AI أو OCR.
- الأسئلة تظهر كبطاقات مستقلة.
- يسمح بتعديل الترجمة والدرجة.
- يسمح بحذف واستعادة السؤال.
- يسمح بإعادة ترتيب الأسئلة.
- القاموس يظهر للمعلم فقط ولا يظهر للطالب.
- التصدير الفعلي مؤجل إلى Phase 1-F.

## قرارات ممنوعة في Phase 1-A

- لا نموذج إجابة.
- لا تصنيف مهارات.
- لا صعوبة.
- لا حسابات دخول.
- لا حفظ دائم.
- لا معالجة PDF حقيقية.
- لا OCR.
