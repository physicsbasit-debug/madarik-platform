# قرارات منصة مدارك

## Phase 1-F2

- تم اعتماد تصدير PDF فعلي كمرحلة مستقلة بعد DOCX.
- PDF يُنشأ عبر ReportLab داخل Backend.
- العربية تُشكّل عبر `arabic-reshaper` و`python-bidi` لأن ReportLab لا يدير RTL تلقائيًا كما يتمنى البشر.
- لا تُرفق أي ملفات خطوط داخل المستودع.
- يعتمد النظام على خطوط النظام مثل DejaVu Sans عند توفرها.
- لا يتم إدراج الصور أو شعار المدرسة في PDF في هذه المرحلة.
- لا يتم إلغاء DOCX؛ يبقى DOCX وPDF متاحين معًا.
