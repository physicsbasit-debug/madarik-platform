# قرارات منصة مدارك

## القرارات المعتمدة

1. اسم المشروع: منصة مدارك.
2. البداية في مواد العلوم العامة والفيزياء والكيمياء والأحياء، مع عدم حصر الهوية في العلوم.
3. التقنية: React + TypeScript للواجهة، Python + FastAPI للخلفية.
4. واجهة Phase 1 متعددة الخطوات:
   - مشروع جديد
   - رفع الملف
   - استخراج الأسئلة
   - مراجعة القاموس
   - مراجعة الأسئلة
   - التصدير
5. الحفظ في Phase 1 مؤقت داخل الجلسة فقط.
6. لا حسابات دخول في Phase 1.
7. التصدير في Phase 1: DOCX وPDF.
8. اتجاه الواجهة والتصدير: RTL.
9. ترقيم الأسئلة: 1، 2، 3، والخيارات: أ، ب، ج، د.
10. نموذج الإجابة مؤجل.
11. النسخة التدريبية المشروحة مؤجلة.
12. قاموس المصطلحات يظهر للمعلم فقط ولا يظهر للطالب في النسخة النهائية.
