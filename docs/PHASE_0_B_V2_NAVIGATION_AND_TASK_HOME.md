# Phase 0-B: V2 Navigation and Task-Oriented Home

## الهدف

تحويل نقطة الدخول من مسار ترجمة واحد إلى مركز مهام واضح لمعلم العلوم.

## المتاح الآن

- ترجمة سريعة
- ترجمة احترافية

## معروض بوصفه قادمًا

- مكتبة المناهج والمصادر
- بنك الأسئلة
- إنشاء اختبار
- أنشطة متمايزة

## ضوابط المرحلة

- البطاقات القادمة تعريفية وغير قابلة للنقر.
- المساران المتاحان يعيدان استخدام النواة الحالية.
- يمكن العودة إلى مركز المهام من مساحة العمل.
- لا تغيير على Backend أو API أو قاعدة البيانات.
