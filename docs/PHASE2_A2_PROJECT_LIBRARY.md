# منصة مدارك - Phase 2-A2 Project Library

## الغرض

بعد إضافة التخزين الدائم في Phase 2-A1، تضيف هذه المرحلة واجهة مكتبة مشاريع محفوظة حتى يستطيع المستخدم فتح مشروع محفوظ أو حذفه من الواجهة.

## ما تم

- إضافة دوال Frontend API:
  - `getProject`
  - `listProjects`
- إضافة مكوّن:
  - `frontend/src/features/project-library/ProjectLibraryPanel.tsx`
- إضافة زر "مكتبة المشاريع" في أعلى الواجهة.
- عرض المشاريع المحفوظة من SQLite.
- فتح مشروع محفوظ.
- حذف مشروع محفوظ.
- إعادة إظهار أزرار Snapshot JSON في أعلى الواجهة:
  - حفظ JSON
  - استيراد JSON

## ما لم يدخل

- حسابات مستخدمين.
- صلاحيات.
- بحث متقدم.
- وسوم أو مجلدات.
- مشاركة مشاريع.
- تكرار مشروع من الواجهة.

## ملاحظات

مكتبة المشاريع الحالية قائمة داخلية بسيطة فوق SQLite. الهدف ليس بناء نظام إدارة ملفات كامل، بل إثبات أن التخزين الدائم صار ظاهرًا للمستخدم ويمكن التعامل معه.
