# Phase 6-C: Assessment Ordering, Sections, and Manual Marks

## الهدف

تحويل مسودة الاختبار من قائمة أسئلة إلى بنية اختبار قابلة للتنظيم.

## الأقسام

يمكن إنشاء أقسام متعددة مثل:

- اختر الإجابة الصحيحة.
- الأسئلة القصيرة.
- المسائل الحسابية.
- الاستدلال والتفسير.

لكل قسم اسم وترتيب مستقل.

## ترتيب الأسئلة

يمكن تحريك كل سؤال للأعلى أو الأسفل. يُحفظ الترتيب داخل المسودة ولا يغير
ترتيب السؤال الأصلي في بنك الأسئلة.

## الدرجات اليدوية

يمكن تعديل درجة السؤال داخل الاختبار فقط. تبقى درجة بنك الأسئلة الأصلية محفوظة،
ويظهر الفرق في واجهة المنشئ.

## التوافق

المسودات القديمة التي لا تحتوي أقسامًا أو إعدادات عناصر تحصل تلقائيًا على قسم
افتراضي وإعداد ترتيب متوافق.

## خارج النطاق

- معاينة ورقة الطالب.
- ترقيم الأقسام في التصدير.
- DOCX/PDF.
- صفحة إجابة منفصلة.
