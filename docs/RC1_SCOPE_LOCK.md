# منصة مدارك - Scope Lock for RC1

## قاعدة عامة

RC1 مرحلة تثبيت، وليست مرحلة توسع. أي ميزة جديدة كبيرة تؤجل إلى Phase 2. نعم، حتى لو كانت مغرية، فالمشاريع لا تنهار من نقص الأفكار بل من زيادتها.

## مسموح داخل RC1

- إصلاح أخطاء واضحة.
- تحسين نصوص صغيرة.
- إصلاح build/test.
- تحسين توثيق القبول.
- إصلاح RTL محدود لا يغير المعمارية.

## ممنوع داخل RC1

- إضافة حسابات.
- إضافة قاعدة بيانات.
- إضافة نموذج إجابة.
- إضافة AI إلزامي.
- تغيير معماري كبير.
- إعادة بناء OCR.
- تغيير مسار التصدير بالكامل.
