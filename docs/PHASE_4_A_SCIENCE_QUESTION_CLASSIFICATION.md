# Phase 4-A: Science Question Classification Foundation

## الهدف

إضافة تصنيف معرفي قابل للمراجعة لكل سؤال علمي.

## الفئات

- `knowledge`: استدعاء أو تعريف أو تسمية أو ذكر.
- `application`: استخدام قانون أو إجراء أو حساب.
- `reasoning`: تفسير أو تبرير أو مقارنة أو تنبؤ أو استنتاج.
- `unclassified`: غياب المؤشرات أو تعارضها.

## الحقول

- الفئة المعرفية.
- درجة الثقة.
- سبب التصنيف.
- مصدر التصنيف.

## التنفيذ

- تصنيف آلي حتمي من شاشة المراجعة.
- تعديل يدوي بواسطة المعلم.
- حفظ الحقول عبر API تحديث السؤال الموجود.
- لا يوجد مزود ذكاء اصطناعي خارجي جديد.

## خارج النطاق

- التصنيف الجماعي.
- ربط السؤال بالمنهج.
- تقدير الصعوبة.
- بنك الأسئلة.
