# Batch 5 — تبسيط غلاف المنصة وإخفاء الإدارة التقنية

تُطبق هذه الدفعة على الفرع الحالي فقط:

`feat/madarik-science-platform-v2`

## الهدف

تقليل الضوضاء المتبقية في رأس المنصة والصفحة الرئيسية دون تغيير Backend أو عقود API.

## التغييرات

- إبقاء التنقل الرئيسي على: الرئيسية، أعمالي، بنك الأسئلة.
- إبقاء «عمل جديد» كإجراء مباشر داخل قائمة المزيد.
- نقل استيراد وتنزيل Snapshot إلى قسم مطوي باسم «إدارة متقدمة».
- تبسيط شريط سياق الوحدة إلى اسم الوحدة واسم العمل فقط.
- إخفاء رسائل نجاح الترجمة والاتصال من الرئيسية.
- إظهار تنبيه فقط عند انقطاع الاتصال أو عدم جاهزية مزود الترجمة.
- إضافة اختبارات حماية للغلاف المبسط.
